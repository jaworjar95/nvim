## Commands
- `:colo` changes the color scheme, use with <Ctrl-d> to display the list of available color schemes
- `:Explore` opens NeoVim built-in file explorer. There are other variants like:
    - `:Sexplore` opens it in a horizontal split, so your file stays visible above/below
    - `:Vexplore` opens it in a vertical split, side by side with your file
    - `:Texplore` opens it in a new tab
    - `:Lexplore` toggles a left-hand sidebar panel (NERDTree-style), and running it again closes it.
